# Titanic Survival Analysis

## Overview
This project analyzes the Titanic dataset and predicts passenger survival using Machine Learning techniques.  
The analysis includes data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling.

The goal of this project is to understand which factors affected passenger survival during the Titanic disaster.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

---

## Dataset
The dataset contains passenger details such as:
- Passenger ID
- Name
- Gender
- Age
- Ticket Class
- Fare
- Cabin
- Embarked Location
- Survival Status

---

## Project Workflow

### 1. Data Collection
- Load Titanic dataset using Pandas
- Understand dataset structure

### 2. Data Cleaning
- Handle missing values
- Remove unnecessary columns
- Convert categorical data into numerical format

### 3. Exploratory Data Analysis (EDA)
- Survival analysis
- Gender-wise survival comparison
- Passenger class analysis
- Age and fare distribution visualization

### 4. Feature Engineering
- Encoding categorical variables
- Feature selection
- Preparing dataset for model training

### 5. Machine Learning Model
The project uses classification algorithms such as:
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors

### 6. Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report

## How to Run the Project

### Clone Repository

```bash
git clone https://github.com/Suhanii13/Titanic-Survival-Analysis.git
```

### Open Project Folder

```bash
cd Titanic-Survival-Analysis
```

### Install Required Libraries

```bash
pip install -r requirement.txt
```

### Run Jupyter Notebook

```bash
jupyter notebook
```

Open the notebook file and run all cells.
